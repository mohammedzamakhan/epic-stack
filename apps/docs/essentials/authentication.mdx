# Authentication

Epic Stack comes with a robust authentication system out of the box. It supports various authentication methods:

## Email/Password Authentication

The classic approach to authentication:

- Secure password hashing with bcrypt
- Email verification flow
- Password reset functionality
- Optional two-factor authentication

## OAuth Authentication

Social login options:

- GitHub OAuth integration
- Easy to add other OAuth providers

## Passkeys

Modern authentication with passkeys (WebAuthn):

- Biometric authentication support
- No more passwords
- Enhanced security

## Sessions

Session management is handled securely:

- HTTP-only cookies
- CSRF protection
- Configurable session duration
- Active session listing and management
