# Styling

Epic Stack uses a modern CSS approach with utility-first styling.

## Tailwind CSS

Tailwind CSS provides:

- Utility-first CSS framework
- Consistent design tokens
- Easy responsiveness
- Dark mode support out of the box

## Component Library

The Epic Stack includes shadcn/ui components:

- Pre-built accessible components
- Customizable through Tailwind
- Based on Radix UI primitives
- Designed for excellent UX and accessibility

## Custom Components

The stack includes custom-built components:

- Form components with validation
- Error boundaries
- Loading indicators
- Toast notifications

## Theme Customization

You can customize the theme through:

- Tailwind configuration
- CSS variables
- Component overrides

## Responsive Design

All components are built with responsive design in mind, ensuring your application looks good on all devices from mobile to desktop.
